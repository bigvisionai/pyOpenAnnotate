## Annotate Images

```
annotate --img <images_directory_path>
```

## Annotate Video
```
annotate --vid <path_to_video_file>
```
## Global Flags
```
-T : View mask window.
```
## Keyboard Navigation
```
N : Save and go to next image
B : Save and go back
C : Toggle clear screen 
T : Toggle mask window
Q : Quit
```
